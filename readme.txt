This plugin adds a feature to a post category to allow the posts in that particular category to be displayed in ascending or descending order by date.

== Description ==

Have you ever wanted to shift the order of your posts to ascending by date opposed to descending? This plugin adds a feature to a post category to allow the posts in that particular category to be displayed in ascending or descending order by date.

When you go to edit your category, you will have 2 options:

* Ascending
* Descending

Which will automatically display posts on that category page ascending/descending based on your specification.

== Installation ==

1. Upload the `ascendingposts` folder into the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Does this plugin sort both ascending and descending =

Yes!
